# ENCODEX

ENCODEX is a terminal-based encoding and decoding tool in **Bash**.  
It supports **Binary, ASCII, Base64, Hex, XOR, Base32, and ROT13**, with a menu-driven interface and color-highlighted outputs.

---

## Features

- Encode/decode text using multiple methods  
- XOR with numeric or single-character key  
- Reusable loop until exit  
- Color-coded output: Cyan → Encoding, Magenta → Decoding  

---
